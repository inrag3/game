# Лабораторная работа №7 - Сетевые игры
Данная лабораторная работа была выполнена в рамках курса CS222 "Компьютерные сети". В этой работе реализована сетевая HTML5 игра с использованием популярного игрового движка Phaser.

## Задача
Моя задача состояла в создании сетевой игры, которая будет использовать HTML5 и Phaser для создания игрового контента и обмена данными между игроками по сети.

## Технология
Для реализации данного проекта были использованы следующие технологии:

- Игровой движок: Phaser
- Библиотека для обмена данными по сети: Socket.io
- Фреймворк для создания веб-приложений: Express

## Файлы проекта
- **index.html**: HTML-страница, содержащая игровое поле и подключающая игровой скрипт.
- **game.js**: Код игры, включающий логику игровых объектов и обмен данных по сети с использованием Socket.io.
- **server.js**: Серверная часть приложения, использующая Express и Socket.io для обработки соединений и обмена данными между игроками.

## Запуск проекта

1. Убедитесь, что у вас установлен Docker и Docker Compose.
2. Склонируйте репозиторий на локальную машину:
```
git clone https://github.com/inrag3/game.git
```
3. Перейдите в директорию проекта:
```
cd game/
```
4. Запустите проект с помощью Docker Compose:
```
docker-compose up --build -d 
```
5. Запустите server.js командой:
```
docker-compose exec node node server.js
```
